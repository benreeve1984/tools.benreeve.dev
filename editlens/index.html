EditLens in the Browser

A research reproduction of Pangram's open-weight EditLens model (Thai et al., ICLR 2026), intended to run entirely client-side in this browser tab.

This is a research reproduction of an AI detection model. AI detectors have documented elevated false-positive rates on non-native English writing and on short text. Do not use these scores to make decisions about other people. The model authors explicitly state these weights should not be used to enforce AI usage policies.

The page shows paragraph-level model output only. There is no document-level score, no export, and no share action. Read the known failure modes, methodology, and validation notes.

Demo mode is active. Scores are fake and only exercise the interface.

Your pasted text is not uploaded. The browser downloads static model files from Hugging Face, caches them when possible, then runs inference locally.
